# SEO Keyword-to-Page Map

## Objective
Rank for high-intent mobile imaging trailer service queries by aligning each keyword cluster to a dedicated page with clear conversion intent.

## Search Landscape Notes
Based on competitive search review, the strongest recurring mobile trailer terms are:
- mobile diagnostic imaging rentals
- mobile imaging equipment rental
- mobile MRI rental
- mobile MRI trailer rental
- mobile MRI scanner rental
- mobile CT scanner rental
- temporary CT scanner rental
- mobile CT trailer rental
- mobile PET/CT rentals
- PET/CT mobile units
- interim mobile imaging
- short-term mobile MRI lease
- long-term mobile imaging trailer lease

The common buying scenarios to address in copy are equipment failure, planned renovation, scanner replacement, patient-volume overflow, rural access, and new imaging program launches.

## On-Page Requirements Per Target Page
- One unique H1 with primary keyword near the start.
- One to two supporting H2 headings with related terms.
- Service proof section with response time and support scope.
- Internal links to /contact and /services.
- FAQ block with concise, direct answers.
- JSON-LD structured data (Service and/or FAQPage).

## Internal Linking Plan
- /services links to all OEM pages.
- OEM pages link back to /services and /contact.
- Home page and relevant sections link to /services using descriptive anchors.

## Measurement Plan (Google Search Console)
- Track query impressions, clicks, CTR, and average position for:
  - /services
  - /services/ge-mobile-mri
  - /services/philips-mobile-mri
  - /services/siemens-mobile-mri
- Review every 14 days:
  - Improve titles/meta descriptions for low CTR queries.
  - Expand copy for queries with impressions but low ranking.
