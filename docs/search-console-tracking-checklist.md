# Search Console Tracking Checklist

## Goal
Measure ranking progress for the new mobile imaging service pages and identify fast optimization wins every 14 days.

## URLs To Track
- /services
- /services/ge-mobile-mri
- /services/philips-mobile-mri
- /services/siemens-mobile-mri
- /services/mobile-mri-trailer-rental
- /services/mobile-ct-trailer-rental
- /services/mobile-pet-ct-trailer

## Query Groups
- Brand + modality:
  - ge mobile mri
  - philips mobile mri
  - siemens mobile mri
- Trailer intent:
  - mobile mri trailer rental
  - mobile mri rental
  - mobile mri scanner rental
  - mobile mri trailer
  - mobile ct trailer
  - mobile pet ct trailer
- Commercial intent:
  - mri trailer rental
  - ct trailer rental
  - mobile imaging trailer

## Setup Steps
1. Submit and verify sitemap.xml in Google Search Console.
2. In Performance, filter by each target page and export query data.
3. Save baseline metrics for clicks, impressions, CTR, and average position.
4. Create one dashboard tab or sheet per URL.

## Review Cadence (Every 14 Days)
1. Identify high-impression queries with low CTR.
2. Update page title and meta description to improve click appeal.
3. Identify queries ranking in positions 8-20.
4. Expand on-page copy with concise sections that match those terms.
5. Add or improve internal links from related pages.

## Action Rules
- If CTR is below 2% and impressions are rising:
  - Improve title wording and meta value proposition.
- If position is 8-20 with growing impressions:
  - Add one focused paragraph and one FAQ item for that query intent.
- If position is below 20 for 30+ days:
  - Build a dedicated support page or location page.

## Monthly Reporting
- Report top 10 gaining queries by clicks.
- Report top 10 pages by impression growth.
- Flag pages with declining CTR or position.
- Record all content changes made so ranking shifts can be attributed.
